Exact contract

Built for unattended agents.

INPUT

One canonical run URL

Both tools accept a public https://github.com/owner/repository/actions/runs/123 URL. Flake classification may also pin an exact attempt.

OUTPUT

Machine-readable decisions

tools/list publishes compact success schemas. Results return identical JSON text and structured content with a stable product identity and service_reuse rule.

SAFETY

No workflow mutation

The server reads bounded public jobs and failed-step logs. It does not rerun workflows, write repository data, expose private runs, or execute repository code.

PAYMENT

Pay only after validation

Malformed or unsupported input fails before payment. Valid calls return an exact x402 v2 Base-USDC requirement; no account or API key is needed.
